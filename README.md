# Pac-Man Game

My first AI-assisted coding project! I built this classic Pac-Man game as a personal side project to get my feet wet with AI-assisted development using Cursor and Claude Code.

## Play It

**[Play Pac-Man in your browser](https://bjonesdataliteracy.github.io/pacman-game/)** — no install needed!

## How to Play
- Use **arrow keys** to control Pac-Man
- Collect all pellets to win
- Grab power pellets to eat the ghosts
- Avoid ghosts or you lose a life (you get 3)

## About This Project
This is a pure HTML5 Canvas game — vanilla JavaScript, no frameworks, no dependencies. It was built as a learning exercise to explore AI-assisted coding workflows with Cursor and Claude Code.
