# Pie Man

A data-literacy-themed Pac-Man game! Navigate the data grid, collect data points, grab golden datasets to clean up dirty data, and avoid Bias, Null, Noise, and Outlier.

My first AI-assisted coding project — a personal side project to get my feet wet with AI-assisted development using Cursor and Claude Code.

## Play It

**[Play Pie Man in your browser](https://bjonesdataliteracy.github.io/pacman-game/)** — no install needed!

## How to Play
- Use **arrow keys** to navigate the data grid
- Collect all **data points** to complete the dataset
- Grab **golden datasets** to temporarily clean up the data villains
- Avoid **Bias**, **Null**, **Noise**, and **Outlier** — or lose your data!

## About This Project
This is a pure HTML5 Canvas game — vanilla JavaScript, no frameworks, no dependencies. Built as a learning exercise to explore AI-assisted coding workflows with Cursor and Claude Code.

Inspired by the mission of helping people learn the language of data and AI.
